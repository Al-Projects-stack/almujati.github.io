# Al Mujati · Portfolio

**Same developer. Different lens.**

A portfolio that explores how the same work can feel completely different depending on how you frame it. Two themes, one body of work — a reminder that perspective shapes perception.

Dual-theme experience: **Cyber Edition** and **Vista Edition**.

## Themes

### Cyber Edition (`/cyber/`)
Dark theme with particle network background, glowing accents, custom cursor, and a terminal-inspired aesthetic. Original portfolio design.

### Vista Edition (`/vista/`)
Full Windows Vista desktop simulation with:
- Aero Glass window effects (backdrop-filter blur, glass borders)
- Draggable, resizable windows with title bar chrome
- Aurora wallpaper (pure CSS/SVG, no external images)
- Desktop icons, taskbar with start menu
- Boot sequence animation
- Right-click context menu on desktop
- Recycle Bin easter egg
- Real-time clock in system tray

## Usage

Open `index.html` to access the theme selector, or navigate directly to:
- `cyber/index.html` for the Cyber portfolio
- `vista/index.html` for the Vista desktop

Theme preference is saved to localStorage on selection.

## Tech

- Pure HTML, CSS, JavaScript — no frameworks, no build tools
- No external dependencies
- Google Fonts for typography
- Deployed on GitHub Pages
